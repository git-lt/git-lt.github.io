MAC终端中安装命令行工具TREE

发表于 2016-03-16   |   分类于 工具   |  

通常我们在使用命令行工具时，习惯性的会查看当前文件下的目录结构和文件，tree命令很好的帮我们解决了这个问题，但是在liunx和mac系统下没有这个命令，在Windows中，系统有自带的tree命令。

两种方式

下面我们在Mac OSX中安装 tree 命令

方法一：我们可以写一个alias 到~/.zshrc下

使用 source .zshrc 更新配置
缺点是没有层级参数配置，会递归显示所有文件，包括隐藏文件

方法二：安装第三方的 tree 命令

修改编译配置

下载解压后，打开目录下的Makefile文件，修改第 55 行，将mac OS X的配置注释打开

配置快捷命令

使用make命令编译（前提是你已经安装了Xcode命令行工具，如果没有，请在终端中执行”xcode-select –install”来安装。）
编译完成后会在目录下生成tree命令的二进制文件，拷贝到/usr/local/bin/目录下

打开 ~/.zshrc 文件，在最后添加别名，保存关闭

使用命令source .zshrc更新配置
然后就可以畅快的使用tree了

常用命令

tree -L 1 显示当前目录下的1级目录
tree -L 2 显示当前目录下的2级目录结构
tree -a 显示当前目录所有的目录结构
